### Manage Your Pills Easily!

Instructions to run this app is as follows
Frontend: 

    cd frontend
    
    yarn install 
    
    yarn start
Backend:

    cd ../backend
    
    pip install -r ./requirements.txt
    
    python manage.py runserver
